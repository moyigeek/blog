## 服务器使用

### 大模型推理作业阶段
安装rust
```shell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
克隆项目
```shell
git clone git@github.com:moyigeek/learning-lm-rs.git
```
进入项目目录
```shell
cd learning-lm-rs
cargo test
```
